# **App Name**: ClassVault

## Core Features:

- Role-Based Content Access: Email-authorized admin dashboard for content management (CRUD) alongside a student-facing interface for browsing and downloading notes.
- Intelligent Study Search: A comprehensive search tool powered by AI that enables students to find relevant notes, chapters, and keywords across multiple subjects efficiently.
- Dynamic Dashboard: A centralized home page featuring search functionality, the latest uploaded notes, subject directories, and live school-wide announcements.
- Password-Protected Premium Content: A feature for gated exam boosters and practice materials that prompts users for a code to unlock high-value resources.
- Subject Directory & Filtering: Hierarchical organization of notes into subjects like Science and Maths with filtering for chapter, status, and upload date.
- Real-time Educational Database: Cloud-hosted database using Firestore for managing structured data for users, subjects, notes, and site-wide announcements.
- Integrated File Viewer & Downloader: Secure interface to open PDF/doc thumbnails and download resources directly to a personal user library.

## Style Guidelines:

- The primary color is a Professional Sapphire Blue (#1750CC) symbolizing trust and clarity in an educational setting.
- The background is a soft, crisp Arctic White (#F8FAFC) to maintain a modern and distraction-free learning environment.
- The accent color is a Bright Azure (#42C2F1), used sparingly for progress bars, buttons, and call-to-action highlights to differentiate interactive elements.
- Headlines and interface labels utilize 'Inter', a grotesque sans-serif for a neutral, machined look. Body text and educational descriptions use 'PT Sans' for added warmth and legibility.
- Clean, thin-stroke geometric icons used to identify subjects and file types, maintaining a professional and non-childish visual hierarchy.
- Mobile-first, card-based layout featuring smooth spacing and 12px rounded corners to emphasize a professional, agency-style finish.
- Minimal page transitions and subtle scale effects on note cards when hovered or selected to provide immediate interactive feedback.
